# **MusicPlayerPlugin**

## 설명

이 플러그인을 이용해 마인크래프트에서 NBS 음악을 플레이할 수 있습니다.
[NoteBlockAPI](https://github.com/koca2000/NoteBlockAPI) 가 필요합니다.

## 명령어
<> : 필수 입력

[] : 선택 입력

/play <파일명> : NBS 파일을 재생합니다.
